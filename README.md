# ⚽ ACE Manager v4.8

**ACE Manager**는 축구 팀 및 선수 데이터를 관리하고 전략적인 인사이트를 제공하는 차세대 축구 관리 시스템입니다. 
FM(Football Manager) 스타일의 유려한 UI와 직관적인 데이터 관리를 제공합니다.

## 🚀 Key Features

- **Ultra-Dark Emerald UI**: 눈이 편안하면서도 프리미엄한 느낌을 주는 다크 모드 인터페이스.
- **FM-Style Onboarding**: 선수 포지션 선정, 역할 부여, 능력치 배분으로 이어지는 체계적인 온보딩 워크플로우.
- **Attribute System**: 100점 캡 시스템 기반의 정교한 선수 능력치 관리.
- **Admin Dashboard**: 팀 전체의 현황을 한눈에 파악하고 관리할 수 있는 강력한 관리자 기능.
- **Local-File Based Persistence**: 별도의 데이터베이스 설정 없이도 안정적인 데이터 보존.

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **State Management**: Local Storage Hooks & Context API
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/needchoi/ace-web-project.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
