# 30. Login Screen (로그인)

## 1. 화면 개요
- 아이디와 비밀번호를 통한 서비스 접속 화면.

## 2. 인증 로직
1.  사용자가 아이디/비밀번호 입력.
2.  Supabase `users` 테이블에서 일치하는 레코드 조회.
3.  **상태 검사 (필수)**:
    - 조회 결과가 없으면: "아이디 또는 비밀번호가 틀렸습니다."
    - 상태가 `PENDING`이면: "관리자의 승인을 기다려주세요."
    - 상태가 `REJECTED`이면: "가입이 거절된 계정입니다."
    - 상태가 `ACTIVE`이면: 로그인 성공 및 대시보드 이동.
