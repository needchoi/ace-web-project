# 32. Admin Page (사용자 승인 관리)

## 1. 화면 개요
- 관리자가 가입 신청자를 확인하고 상태를 변경하는 관리 전용 화면.

## 2. 주요 기능
- **신청자 목록 조회**: `status === 'PENDING'`인 사용자 리스트업.
- **상세 정보 확인**: 신청자의 이름, 아이디, 생년월일, 신청 시간 표시.
- **승인 액션**: '승인' 클릭 시 해당 사용자의 `status`를 `'ACTIVE'`로 변경.
- **거절 액션**: '거절' 클릭 시 해당 사용자의 `status`를 `'REJECTED'`로 변경.

## 3. 접근 제어
- `role === 'ADMIN'` 권한을 가진 사용자에게만 메뉴 노출 및 접근 허용.
