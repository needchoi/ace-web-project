# 31. Signup Screen (회원가입/가입신청)

## 1. 화면 개요
- 서비스 이용을 위한 초기 데이터 입력 및 가입 신청 화면.

## 2. 주요 입력 항목 (UI 컴포넌트)
- **아이디**: 중복되지 않는 고유 식별자 (Text Input)
- **비밀번호**: 8자 이상의 영문/숫자 조합 (Password Input)
- **이름**: 실명 입력 (Text Input)
- **생년월일**: 날짜 선택 (Date Picker/Input)

## 3. 로직 (Logic)
- **신청 액션**: 입력값 유효성 검사 후 Supabase `users` 테이블에 데이터 삽입.
- **기본값 설정**: `status` 필드는 시스템에 의해 자동으로 `'PENDING'`으로 설정됨.
- **완료 안내**: "가입 신청이 완료되었습니다. 관리자 승인 후 이용 가능합니다." 메시지 노출 후 로그인 화면으로 이동.
