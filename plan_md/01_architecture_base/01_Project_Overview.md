# 01. Project Overview (프로젝트 개요)

## 1. 프로젝트 비전
**ACE (Athletic Club Excellence)**는 가입이나 로그인 절차 없이 누구나 즉시 접근할 수 있는 오픈형 축구팀 매니지먼트 플랫폼입니다.

## 2. 핵심 아키텍처 원칙
- **Zero Authentication**: 번거로운 인증 절차(회원가입, 로그인)를 완전히 제거하여 접근성을 극대화합니다.
- **Open Collaboration**: 팀원 누구나 접속 즉시 팀의 일정과 경기 결과를 확인하고 참여할 수 있습니다.
- **Public Data via Supabase**: Supabase는 공용 데이터(Public Data)를 저장하고 실시간으로 동기화하는 개방형 데이터 저장소로 활용됩니다.

## 3. 주요 기능
- 로그인 없이 바로 진입하는 오픈 대시보드.
- 팀 공통 경기 일정 및 데이터 실시간 확인.
- 빠르고 직관적인 모바일 친화적 인터페이스.
