# 01. Project Overview (프로젝트 개요)

## 1. 프로젝트 비전
**ACE (Athletic Club Excellence)**는 축구팀의 선수 데이터를 체계적으로 관리하고, 철저한 권한 기반의 승인 절차를 거쳐 운영되는 차세대 축구팀 매니지먼트 플랫폼입니다. FM(Football Manager) 스타일의 유려한 UI와 직관적인 데이터 관리를 제공합니다.

## 2. 핵심 아키텍처 원칙
- **Strict Authentication**: 관리자(Admin)의 철저한 승인 과정을 통과한 일반회원만이 시스템에 접근할 수 있습니다.
- **Role-based Access Control**: 가입 대기, 일반회원, 관리자, 거절 등의 계정 상태에 따라 접근 권한과 기능이 철저히 분리됩니다.
- **Centralized Admin Dashboard**: 관리자는 단일 대시보드에서 가입 신청을 처리하고 팀 전체 데이터를 관리합니다.

## 3. 주요 기능
- 회원가입 신청 및 대기 시스템
- 관리자의 가입 승인/거절 워크플로우
- 로그인 성공 시 진입하는 데이터 대시보드
- 빠르고 직관적인 모바일 친화적 다크 모드 인터페이스
